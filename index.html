PEPSI SPIRE

Pepsi is the world’s leading beverage company, and one of the largest manufacturer, distributor and marketer of nonalcoholic beverage concentrates the syrups in the world. Pepsi Organization has installed a machine called Spire at few locations in US from which user can selects the beverages and its flavors, Spire Machines has basic beverage brands like Pepsi, Mountain Due etc and flavors like chary, vanilla, grape etc. Present system work is totally manual. We have automated this system with the help of Mobile App. In Mobile App user has already details of brands of beverages as well as flavors. Now user can make his choice of drink in the application. Once he reach at store then user send this information to Spire Machine using BLE (BlueTooth Low Energy​) support. iOS Application using the the CoreBlueTooth framework to Scan, Connect and Search of Beverage on Machine and Send the Details of choice of drink from Mobile application to Spire Machine. User can give the Pour command from iOS Application.
